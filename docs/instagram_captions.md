## API Key Note
The `tools/draft_social_post.py` tool is set up to auto-generate future captions.
To activate it, open `.env` and add your Anthropic key after the `=` sign:
```
ANTHROPIC_API_KEY=sk-ant-...
```
Get your key at: console.anthropic.com → API Keys
